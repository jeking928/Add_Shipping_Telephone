PURPOSE & AIM

This plugin adds the telephone number to the Shipping Address of the Customer's Order. You can add an address during checkout or from their account page.

INSTALLATION

!!!!!BACKUP YOUR STORE BEFORE YOU BEGIN!!!!!

Installation

  1. Copy and paste the ADD_TELEPHONE_SQL.sql file in the Admin Tools Install SQL Patch screen and press send.
  2. Rename YOUR-ADMIN to match your store admin directory name.
  3. Rename YOUR_TEMPLATE to match your store template directory name.
  4. Upload all files merging any changes you have made to core files.

Usage

No specific usage instructions. The telephone number field will be displayed and required when a shopper creates a new address during checkout or from the My Account page.
